# 🏠 House Price Prediction System
## 📌 Overview
The **House Price Prediction System** is a machine learning project that predicts the price of a house based on key features such as area, number of bedrooms, bathrooms, location, and other property attributes.
This project demonstrates an **end-to-end regression pipeline** including data preprocessing, model training, evaluation, and deployment using a simple user interface.
## 🚀 Features
* 🤖 **Machine Learning Regression Model**
  * Predicts house prices accurately
  * Trained on structured housing dataset
* 📊 **Data Visualization**
  * Price distribution analysis
  * Feature correlation heatmaps
  * Area vs price trends
* 🧠 **Feature Engineering**
  * Encoding categorical variables
  * Handling missing values
  * Scaling numerical features
* 🎯 **Real-Time Prediction**
  * User inputs house details
  * Instant price prediction output
* 📁 **Scalable Pipeline**
  * Easy to extend with new features
  * Modular ML workflow
## 📊 Dataset
The dataset contains housing-related features such as:
* Area (sq ft)
* Number of bedrooms
* Number of bathrooms
* Floors
* Parking availability
* Furnishing status
* Location
* Age of property
* Price (Target variable)
👉 You can use:
* Kaggle House Price datasets
